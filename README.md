## ✨ Features

- ⚡ **Generate in seconds** — enter username, click, done
- 🎨 **4 themes** — Default, Dark, Cyberpunk, Ocean
- 📊 **Live GitHub stats** — followers, repos, stars, streaks
- 🔤 **Top languages** — auto-detected from your repos
- 🚀 **Featured projects** — top starred repos shown automatically
- 📥 **One-click download** — save as README.md
- 👀 **Live preview** — see how it looks before using
- 📱 **Mobile responsive** — works on any device

## ⚡ Quick Start

```bash
# Clone the repo
git clone https://github.com/basitali08/profile-readme-generator.git
cd profile-readme-generator

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

---

## 🛠️ How It Works

```
1. Enter a GitHub username
2. App fetches data via GitHub API
3. Stats are compiled (followers, repos, languages)
4. Top projects are selected by stars
5. README is generated with 4 theme options
6. Preview shows instantly
7. Download as README.md
```
